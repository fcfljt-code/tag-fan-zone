7. Einstellungen

Profil bearbeiten

8. Fangruppe-Bewerbungen

So funktioniert es

  1. Ein Besucher füllt das Bewerbungsformular aus
  2. Beim Absenden öffnet sich sein E-Mail-Programm
  3. Er muss die E-Mail manuell senden
  4. Du erhältst die E-Mail an deine eingestellte Adresse
Wichtig: Die E-Mail wird NICHT automatisch gesendet! Der Besucher muss in seinem E-Mail-Programm auf "Senden" klicken.

9. App installieren (PWA)

Die Website ist eine Progressive Web App (PWA) und kann wie eine echte App installiert werden.

Auf Android

  1. Öffne die Website in Chrome
  2. Tippe auf das Menü (drei Punkte oben rechts)
  3. Wähle "Zum Startbildschirm hinzufügen"
  4. Bestätige mit "Hinzufügen"

Auf iPhone/iPad

  1. Öffne die Website in Safari
  2. Tippe auf das Teilen-Symbol (Quadrat mit Pfeil)
  3. Scrolle nach unten und tippe auf "Zum Home-Bildschirm"
  4. Bestätige mit "Hinzufügen"

Auf Windows/Mac

  1. Öffne die Website in Chrome oder Edge
  2. Klicke auf das Install-Symbol in der Adressleiste
  3. Bestätige die Installation
Vorteile: Startet ohne Browser-Leiste, funktioniert offline, schnellerer Zugriff, eigenes App-Icon.

10. Veröffentlichung auf GitHub Pages

Schritt 1: GitHub Account erstellen

Gehe zu github.com und erstelle einen kostenlosen Account (falls noch nicht vorhanden).

Schritt 2: Neues Repository erstellen

  1. Klicke auf "+" → "New repository"
  2. Name: z.B. tsg-hoffenheim-fan
  3. Wähle "Public"
  4. Klicke auf "Create repository"

Schritt 3: Dateien hochladen

  1. Im Repository, klicke auf "uploading an existing file"
  2. Ziehe alle Dateien aus dem Projekt-Ordner hinein
  3. Klicke auf "Commit changes"

Schritt 4: GitHub Pages aktivieren

  1. Gehe zu "Settings" (Zahnrad)
  2. Klicke links auf "Pages"
  3. Unter "Source": Wähle "Deploy from a branch"
  4. Unter "Branch": Wähle "main" und "/ (root)"
  5. Klicke auf "Save"
Fertig! Nach 1-2 Minuten ist deine Website unter dieser Adresse erreichbar:

https://DEIN-USERNAME.github.io/tsg-hoffenheim-fan/

Wichtig: Service Worker anpassen

Öffne index.html und ändere diese Zeile:

navigator.serviceWorker.register('/service-worker.js')

zu:

navigator.serviceWorker.register('./service-worker.js')

Technische Hinweise

Datenspeicherung

Alle Daten werden im localStorage des Browsers gespeichert.

Achtung: Wenn der Browser-Cache gelöscht wird, gehen die Daten verloren! Erstelle regelmäßig Backups.

API-Daten

Die Bundesliga-Tabelle und der Spielplan werden von OpenLigaDB geladen (kostenlos, ohne API-Key). Funktioniert nur mit Internetverbindung.
